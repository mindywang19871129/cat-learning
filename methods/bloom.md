# 布鲁姆掌握学习 — 严控进度

## 核心理念
每个知识点必须**真正掌握**才能进入下一个。宁可慢，不可躁。基础不牢，地动山摇。

## 掌握标准

| 掌握度 | 状态 | 措施 |
|--------|------|------|
| 90-100 | ✅ 已掌握 | 降低频率，定期复习即可 |
| 60-89 | 📖 学习中 | 正常节奏，持续练习 |
| 30-59 | ⚠️ 薄弱 | 增加练习，降低难度，暂停新内容 |
| 0-29 | 🔴 严重不足 | 停止新内容，回炉基础 |

## 进度控制规则

### 铁律1：知识点不掌握不开新
- 任一正在学习的知识点 mastery < 90，不得引入该领域的全新知识点
- 但可以在同知识点下出变式题巩固

### 铁律2：错题未订正不开新课
- 当前错题本中有未订正的错题 → 优先订正
- 订正标准：同类变式题连续2次正确 → 错题标记为已订正

### 铁律3：逐日完成
- 前一天的题目未完成批改 → 今日不出新题
- 已批改但未订正完 → 今日先订正再出新题

## 学习节奏

```
出题 → 答题 → 批改 → 订正 → 确认掌握 → 开启下一知识点
                                              ↓
                                          （循环）
```

## 正向评价
掌握学习不追求速度，追求掌握质量。评价时：
- 强调进步而非排名："比上周多掌握了2个知识点！"
- 强调理解而非速度："没关系，我们慢慢来，理解最重要"
- 强调过程而非结果："你认真思考的样子很棒！"
