# 皮亚杰认知发展理论 — 具象优先

## 核心理念
小学三年级学生（8-9岁）处于**具体运算阶段**，思维需要借助具体事物、图画和生活经验，抽象思维尚未成熟。

## 教学原则

### 1. 先具象后抽象
- 每教一个新概念，先举3个生活中的具体例子
- 用实物、图画、故事辅助理解
- 再用例子总结出知识点

### 2. 生活化教学示例

**数学示例：**
- 教分数 → "妈妈把披萨切成8块，你吃了3块，吃了披萨的几分之几？🍕"
- 教面积 → "你的书桌有多大的桌面？用课本能不能铺满？"
- 教除法 → "15颗糖分给3个小朋友，每人几颗？"
- 教乘法 → "一盒水彩笔12支，3盒一共多少支？"

**英语示例：**
- 教颜色 → 用彩虹🌈图：Red, Orange, Yellow, Green, Blue, Indigo, Violet
- 教食物 → 用冰箱图：milk in the fridge, apple on the table
- 教动词 → 用表情包配动作：😊 smile, 😢 cry, 🏃 run

### 3. 避免抽象
❌ "分数的分子表示取的份数"  
✅ "把蛋糕切成8份，你拿了3份，写成数学就是 3/8"

❌ "area = length × width"  
✅ "数一数这个长方形能被多少个1平方厘米的小方格铺满？"

### 4. 多用直观工具
- 用数轴解释数的大小
- 用方格纸理解面积
- 用实物图理解英语词汇
- 用对比图理解语法时态
