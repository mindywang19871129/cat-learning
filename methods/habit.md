# 习惯养成 — 正向激励

## 核心理念
小学阶段最重要的不是学了多少知识，而是养成了什么样的学习习惯。  
良好的习惯需要：**合理的节奏 + 持续的正向反馈 + 自主感的培养**。

## 学习时长控制

### 单次学习不超过25分钟
- 9岁以下注意力集中时间约15-20分钟
- 每25分钟一个"番茄钟"，然后休息5分钟
- 每日总学习量控制在2-4个番茄钟

### 每日出题量控制
- 平日：数学4道 + 英语4道 = 约20分钟
- 周末：可适当增加，但不超过平日的1.5倍
- 周五特殊：推送三天题目（周五+六+日），但每天独立计时

## 正向激励体系

### 😺 小肥猫的鼓励层级

**基础鼓励（每次都用）：**
- "你今天完成了学习任务，为自己鼓掌！👏"
- "每道题都在认真思考，小肥猫看到了！"
- "今天的你有比昨天的你进步一点点吗？肯定有！"

**进步鼓励（连续正确时）：**
- "连续3题全对！你找到感觉了！🌟"
- "上周错过的题这周全对了，这就是坚持的力量！"
- "难度升级了你也做对了，升级成功！🎉"

**努力鼓励（遇到困难时）：**
- "这道题有点难，但你还没放弃，这就是最棒的品质！"
- "错了没关系，你认真思考的过程比答案更重要"
- "我看到了你在努力，慢慢来，我陪着你！"

### 学习里程碑
- 连续7天完成学习 → 🏅 "坚持之星"
- 掌握10个知识点 → ⭐ "知识达人"
- 错题本清零一次 → 🎯 "纠错勇士"
- 完成一次模考 → 🎓 "勇敢挑战者"

## 培养自主习惯

### 不替代，只引导
- 不主动催促："该学习了！"
- 改为提醒："今天的探索任务准备好了，你准备好了吗？"
- 让孩子自己选择先做数学还是先做英语

### 学习日志可视化
```
📊 本周学习报告
完成天数: 7/7 ✅
数学正确率: 85% 📈 (+5%)
英语正确率: 78% 📈 (+3%)
最棒进步: 面积计算从不会到全对！
待巩固: 英语时态题需要再练练
```

### 规律作息
- 固定推送时间（默认09:00）
- 建议学习时段：09:00-09:25
- 提供"今日已完成"标记仪式感

## 家长配合建议

- 给孩子一个安静的学习角落
- 在学习时间减少打扰
- 完成后真诚地表扬努力（不是聪明）
- 不在批改结果出来前主动辅导
- 让孩子自己面对错题，只在小肥猫引导不够时补充
