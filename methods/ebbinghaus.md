# 艾宾浩斯遗忘规律 — 科学复习

## 核心理念
遗忘是有规律的：学习后20分钟遗忘42%，1小时遗忘56%，1天遗忘74%。  
**对策：** 在遗忘临界点之前及时复习。

## 复习时间节点

| 复习次数 | 间隔 | 说明 |
|----------|------|------|
| 第1次复习 | 当日 | 当天学完立即复习 |
| 第2次复习 | 次日（1天后） | 第二天回想一下 |
| 第3次复习 | 4天后 | 差不多一周内 |
| 第4次复习 | 7天后 | 一周后再巩固 |
| 第5次复习 | 15天后 | 半个月最终确认 |

## 出题中的复习穿插

### 复习题自动插入逻辑
1. 系统每天扫描错题本
2. 找到距离上次复习达到对应间隔的错题
3. 生成同知识点的变式题
4. 插入到当日练习中（每天至少1道复习题）

### 复习优先级
1. **最高优先级：** 多次犯错的题目（错误次数 ≥ 2）
2. **高优先级：** 即将遗忘的内容（距上次复习4-7天）
3. **中优先级：** 薄弱知识点（mastery < 60）的关联题
4. **低优先级：** 正常遗忘周期的常规复习

## 复习方式

不是简单重复原题，而是：
- **数学：** 同知识点变式题（换数字、换场景）
- **英语：** 同词汇不同语境、同语法不同句子
- **标记：** 在题目上标注 🐱复习提醒，让孩子知道这是巩固

## 遗忘曲线追踪

系统记录每个错题的复习历史：
```json
{
  "question_id": "M20260514_03",
  "error_dates": ["YYYY-MM-DD", "YYYY-MM-DD"],
  "review_dates": ["YYYY-MM-DD", "YYYY-MM-DD", "YYYY-MM-DD"],
  "current_review_count": 2,
  "next_review_date": "YYYY-MM-DD（当天日期）"
}
```

复习后正确 → review_count + 1，进入下一个间隔
复习后仍然错误 → review_count 重置为 0，重新开始周期
