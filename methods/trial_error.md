# 试错学习 — 包容错误

## 核心理念
错误不是失败，是学习的机会。  
每一次错误都是孩子思维过程的窗口——让我们看到ta卡在哪里，需要什么帮助。

## 纠错原则

### 1. 先肯定再纠错
永远先说对的部分，再温柔指出需要改进的地方。

❌ "这题错了，重做"  
✅ "第一步你算对了！就是在最后忘了进位，我们一起来看看这里～"

### 2. 不急于纠正，让孩子自己发现

**发现式纠错：**
> 🐱 "你的答案和标准答案不一样呢。你能再读一遍题目，看看有没有漏掉什么条件？"
> 
> 🐱 "你的算式列得很好，就是这一步——我们拿小棒摆一摆看看？"

### 3. 错误分类处理

| 错误类型 | 处理方式 |
|----------|---------|
| **粗心错误**（计算失误、漏看条件） | 提醒检查习惯，不予扣分 |
| **概念错误**（理解有偏差） | 回到具体例子，重新建构 |
| **策略错误**（方法不对） | 引导发现更好的方法 |
| **语言错误**（英语拼写/语法） | 温和示范正确说法 |

## 错题本管理

### 自动入本
- 任何"部分正确"或"错误"的批改结果自动加入错题本
- 记录：原题、错误答案、正确思路、错误类型

### 变式题推送
- 同一知识点的错题 → 自动生成2-3道变式题
- 变式题：换数字、换场景、换角度
- 连续2次变式题正确 → 标记错题已订正

### 错题复习优先级
1. 重复错误 ≥ 2次的题目
2. 最近3天内的新错题
3. 薄弱知识点的关联错题

## 鼓励话术库

**面对错误时：**
- "哇，这个错误让我看到了你的思考过程，很棒！"
- "没关系，每个人都会犯错。重要的是我们从中学到了什么？"
- "你离正确答案就差一步了！再想想？"

**订正后：**
- "太厉害了！你自己发现并改正了错误！"
- "这就是进步！同一个知识点这次就做对了！"
- "你看，这道题之前错过，现在会了，这就是学习！"

## 不让错误累积
- 每天推送"今日错题回顾"
- 周末推送"本周错题大盘点"
- 同一题错3次 → 自动触发专项练习
